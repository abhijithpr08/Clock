# Analog & Digital Clock Project

This project is a simple real-time clock created using HTML, CSS, and JavaScript.  
It displays an analog clock with rotating hands along with a digital clock showing the current time in 12-hour format with AM/PM.

---

## Features

- Live working analog clock
- Digital clock showing time with seconds and AM/PM
- Auto-updates every 1 second
- Smooth hand movement using CSS rotation
- Clean modern design
- Mobile-friendly and responsive

## How This Project Works

JavaScript:

- The `setInterval()` function updates the clock every second.
- The `Date()` object gets the current hour, minute, and second.
- Clock hand positions are calculated using ratios and angles.
- The digital display shows formatted time with AM/PM.

CSS:

- The clock design uses circles and hand shapes.
- Hands rotate using CSS variables.
- Media queries resize everything for mobile screens.

HTML:

- Clock layout is structured using `div` elements.
- Numbers from 1 to 12 are placed around the clock face.
- A digital time container is placed beside the analog clock.
