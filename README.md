# Analog & Digital Clock Project

This project is a simple real-time clock created using HTML, CSS, and JavaScript.  
It displays an analog clock with rotating hands along with a digital clock showing the current time in 12-hour format with AM/PM.

## Features

- Live working analog clock

- Smooth hand movement using CSS rotation  
- Clean modern design  
- Mobile-friendly and responsive
